# Supplementary data for the manuscript "Sample-specific protein-protein interaction networks inferred from transcriptomics and proteomics show high similarities"

We generated sample-specific protein-protein interaction networks from both transcriptomic and proteomic data. The used omics data comes from a multiomics framework meaning that the transcriptome and the proteome were measured in the same samples. We performed a comparative analysis of the inferred networks. Here, we provide the code for the generation of the network and the performed analyses for reproducibility. The generated networks as well as their calculated characteristics can be found on Zenodo: [10.5281/zenodo.10405106](https://doi.org/10.5281/zenodo.21129948)
